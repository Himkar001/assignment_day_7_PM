# Personal Finance Calculator 💰

A Python-based personal finance calculator built for an AI startup’s
employee benefits portal.

## Objective
Collect employee salary details and generate a validated, professional
financial breakdown including tax, rent, savings, and disposable income.

## Features
- Input validation for all numeric fields
- Monthly salary, tax, savings, and rent calculations
- Annual financial projections
- Clean, formatted console output
- PEP 8 compliant Python code

```bash
python src/finance_calculator.py
